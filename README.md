# Ranljivosti Spletne Strani

Projekt je namenjen analizi, demonstraciji in testiranju pogostih varnostnih ranljivosti na spletnih aplikacijah (npr. OWASP Top 10 ranljivosti, kot so SQL injekcije, XSS, CSRF ali napačne konfiguracije).

## Vsebina
- Primeri ranljive kode
- Navodila ali skripte za testiranje ranljivosti (PoC)
- Predlogi in popravki za varno kodiranje

- # Šifriranje in Kriptografija

Ta mapa vsebuje implementacije in primere različnih kriptografskih algoritmov za zaščito podatkov (simetrično in asimetrično šifriranje).

## Vsebina
- Implementacija algoritmov (npr. AES, RSA, DES ali klasične šifre)
- Skripte za šifriranje (encryption) in dešifriranje (decryption) podatkov/teksta

- # Varnost - Naloga 2

Študijski projekt ali praktična naloga, ki obravnava specifičen problem s področja sistemske ali aplikacijske varnosti.

## Vsebina
- Izvorna koda naloge
- Konfiguracijske datoteke ali testni primeri

- # Zgoščevalne Funkcije (Hashing)

Projekt prikazuje delovanje zgoščevalnih (hash) algoritmov, ki se uporabljajo za zagotavljanje integritete podatkov in varno shranjevanje gesel.

## Vsebina
- Primeri uporabe algoritmov (npr. SHA-256, MD5, bcrypt)
- Primerjava hitrosti, varnosti in generiranje zgoščenih vrednosti
